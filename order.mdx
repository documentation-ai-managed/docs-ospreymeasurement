<Callout kind="info" collapsed="false">
  A reference sensor is included as part of the system - whether standard build or custom build - and should not be counted when ordering the required quantity of measurement sensors.&#x20;
</Callout>

As discussed in the [Designing](/designing) section, the ASP is supplied either as a standard build, where each sensor is at a standard spacing, or a custom build in which each sensor is custom built to the customer's specification.&#x20;

## Part Numbers

<Columns cols="2">
  <Card title="Standard Build" icon="align-justify-icon" horizontal="false" target="_self">
    Standard builds come with sensors at 1m, 2.5m, 5m or 10m spacings:

    **ASP-ST**

    Standard terminations for an ASP system. This includes terminations, reference sensor and 10m cable

    **ASP-###**

    Settlement sensors including cable, hose and hydraulic fluid for set distances:

    - ASP-010 - 1m

    - ASP-025 - 2.5m

    - ASP-050 - 5m

    - ASP-100 - 10m

    ### Example

    1 x 45m ASP with sensors at 2.5m spacing:

    ASP-ST x 1

    ASP-025 x 18&#x20;
  </Card>

  <Card title="Custom Build" icon="align-center" horizontal="false" target="_self">
    Custom builds are build to customer specification. Complete the ASP custom build form by entering the distance of each sensor from the reference sensor.

    **ASP-CBU**

    Custom built ASP, price per unit. Includes terminations, reference sensor and 10m cable

    **ASP-CBM**

    Custom built ASP, price per metre. Includes hose, cable and hydraulic fluid.

    **ASP-CBS**

    Custom built ASP, price per sensor. Includes settlement sensor at customer specified position within the system.

    ### Example

    1 x ASP with 6 sensors, with the farthest sensor 95m from the reference sensor:

    ASP-CBU x 1

    ASP-CBM x 95

    ASP-CBS x 6
  </Card>
</Columns>
