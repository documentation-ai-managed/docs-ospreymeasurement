title: Data Management
description: Convert ASP sensor outputs to relative or absolute settlement
---
The ASP nodes carry out onboard conversion of pressure to mm head. Change in elevation is relative to the reference sensor(s). To calculate the current elevation of a measurement node, the following calculations are used:

### Change in Relative Elevation

#### Fixed Reference Sensor

If the elevation of the reference sensor is assumed to be fixed:

```typescript
Δeₙ = Δₙ - Δᵣ
```

**Where:**

- `Δeₙ` = change in relative elevation of a given node

- `Δₙ` = n₁ - n₀ = current reading at node n minus initial reading at node n

- `Δᵣ` = r₁ - r₀ = current reading at reference sensor, or the mean of readings from multiple reference sensors minus initial reference reading

#### Derived Reference

If the elevation of the reference is derived:

```typescript
Δeₙ = Δₙ - Δᵣ + Δd
```

**Where:**

- `Δeₙ` = change in relative elevation of a given node

- `Δₙ` = n₁ - n₀ = current reading at node n minus initial reading at node n

- `Δᵣ` = r₁ - r₀ = current reading at reference sensor, or the mean of readings from multiple reference sensors minus the initial reading at the reference sensor(s)

- `Δd` = d₁ - d₀ = current reading of the external reference minus the initial reading

### Absolute Elevation

To calculate the absolute elevation of a measurement node:

```typescript
Eₙ₁ = Δeₙ + Eₙ₀
```

**Where:**

- `Eₙ₁` = Current reduced level of a given node

- `Δeₙ` = change in relative elevation as described above

- `Eₙ₀` = Initial reduced level of node at time of installation

<Callout kind="tip" collapsed="false">
  Always record initial elevations (Eₙ₀) during installation and the baseline reading period to enable accurate absolute elevation calculations.
</Callout>
