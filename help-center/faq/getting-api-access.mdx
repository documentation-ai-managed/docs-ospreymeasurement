## Overview

You need an API key to use Ospreymeasurement's data services.

## Steps

1. Login to your dashboard at `https://ospreymeasurement.com/dashboard`
2. Navigate to the API section
3. Click "Generate API Key"
4. Copy and save your key securely

## Example Usage

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" https://api.ospreymeasurement.com/v1/measurements
```
